# 옥모닝 고객지원

옥모닝 사용 중 문제가 있거나 문의사항이 있으시면 아래 이메일로 연락해 주세요.

**이메일**: <secalabs1@gmail.com>

영업일 기준 1~3일 이내에 회신을 드립니다.

## 자주 묻는 질문

### 알람이 울리지 않아요

- 배터리 최적화 예외 설정 (Android: 설정 → 배터리 → 배터리 사용량 최적화 → 옥모닝 제외)
- 알림 권한 허용 확인
- 정확한 알람 권한 허용 확인 (Android 12 이상)

### 구독을 해지하고 싶어요

- **Android**: Google Play → 메뉴 → 구독 → 옥모닝 → 해지
- **iOS**: 설정 → Apple ID → 구독 → 옥모닝 → 해지

### 환불 받고 싶어요

환불은 Google Play / App Store의 정책에 따라 진행되며, 해당 플랫폼의 고객센터를 통해 신청해 주세요.
