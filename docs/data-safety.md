# 옥모닝 — Play Console 데이터 안전 양식 답변 초안

검토일: 2026-05-15. Play Console → 앱 콘텐츠 → 데이터 안전 메뉴에서 그대로 응답하면 된다.
*실제 통신 행동은 출시 전 패킷 캡쳐/RevenueCat 대시보드로 한 번 재확인 권장.*

## 1. 데이터 수집 여부

**예 (Yes)** — 제3자 결제 사업자(Google Play Billing) 및 RevenueCat을 통해
구매 정보를 처리하기 때문에 "데이터를 수집한다"고 응답해야 한다.

## 2. 수집/공유하는 데이터 항목

| 데이터 유형 | 항목 | 수집 | 공유 | 목적 | 선택/필수 |
|---|---|---|---|---|---|
| 금융 정보 | 구매 내역 | 수집 | 공유 | 앱 기능 (구독 자격 확인) | 필수 (구독 시) |
| 앱 활동 | 앱 내 검색, 사용자가 생성한 콘텐츠(알람 설정) | **수집 안 함 (서버 미전송)** | — | — | — |
| 앱 정보 및 성능 | 진단 정보 (크래시 로그) | 수집하지 않음 (현재 분석 SDK 없음) | — | — | — |
| 기기 또는 기타 ID | RevenueCat anonymous app user ID | 수집 | 공유 | 앱 기능 (구독 추적) | 필수 (구독 시) |
| 음성 | 마이크 입력 | **로컬 처리만**. `speech_to_text` 패키지를 통해 OS의 음성인식 API로 전달됨. 텍스트 결과만 앱에서 사용하며 옥모닝 서버로는 전송하지 않음 | OS 제공자 (Google STT)에 전달될 수 있음 | 앱 기능 (말하기 퀴즈 채점) | 선택 (말하기 미션 사용 시만) |
| 위치 / 연락처 / 캘린더 / 사진 / 신원 | 일체 수집하지 않음 | — | — | — | — |

> ⚠️ 안드로이드의 `SpeechRecognizer`는 단말의 Google 음성인식 서비스를 호출하며,
> 단말 OS가 해당 발화를 Google 서버로 전송할 수 있다. 옥모닝 자체는 Google 외부로
> 음성을 송출하지 않지만, 데이터 안전 양식에는 "공유될 수 있음"으로 명시하는 것이 안전.

## 3. 보안 관행

- ✅ **전송 중 암호화**: 모든 외부 통신(Google Play Billing, RevenueCat)은 HTTPS로 처리됨.
- ✅ **사용자가 데이터 삭제를 요청할 수 있음**: 구독 데이터는 RevenueCat 측에서 삭제 요청 가능. 로컬 데이터는 앱 삭제 시 모두 제거됨.
- ✅ **자발적 의견에 따른 보안 검토**: 출시 전 자체 권한 검토 완료.

## 4. 광고 ID

- 광고 라이브러리 미통합 → **광고 ID 사용 안 함**.

## 5. 어린이 대상 여부

- 만 13세 미만을 직접 대상으로 하지 않음 → "예/아니오" 질문에 **아니오** 응답.
- 단, 모든 연령대가 사용 가능하므로 콘텐츠 등급 설문에서 "전체 이용가" 유지.
