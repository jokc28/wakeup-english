## 1. 수집하는 개인정보 항목

회사는 옥모닝 서비스(이하 "서비스") 제공을 위해 다음 정보를 처리합니다.

### 1) 단말 내부에만 저장되는 정보
- 알람 설정값 (시각, 반복 요일, 미션 유형)
- 학습 진도 (정답률, 누적 XP, 마스터한 표현 목록)
- 무료 체험 시작 시각 (`SharedPreferences`)

### 2) 외부로 전송되는 정보
- **구독 결제 정보**: Google Play Billing 및 RevenueCat에서 처리. 회사는 결제수단
  자체(카드 번호, 계좌번호 등)에 접근하지 않습니다. RevenueCat은 익명 식별자
  (App User ID), 구매 시각, 구독 상태를 저장합니다.
- **마이크 입력**: 사용자가 말하기 미션을 수행할 때 단말의 운영체제 음성인식
  서비스(Android `SpeechRecognizer` / iOS `SFSpeechRecognizer`)로 전달되며, 운영체제
  제공사 정책에 따라 처리됩니다. 회사 서버로는 송출되지 않습니다.

## 2. 개인정보의 처리 목적

- 알람 동작 및 학습 콘텐츠 제공
- 유료 구독 자격 확인 및 영수증 검증
- 환불·고객문의 응대

## 3. 개인정보의 보유 및 이용기간

- 단말 내부 저장 정보: 사용자가 앱을 삭제할 때까지.
- 결제 관련 정보: RevenueCat 및 Google Play Billing의 보관 정책에 따름.

## 4. 개인정보의 제3자 제공

회사는 다음 경우 외에는 개인정보를 제3자에게 제공하지 않습니다.

| 제공받는 자 | 제공 항목 | 제공 목적 |
|---|---|---|
| Google LLC (Google Play Billing) | 결제 식별자 | 결제 처리 |
| RevenueCat, Inc. | 익명 App User ID, 구매 이벤트 | 구독 상태 추적 |
| Google LLC (Speech-to-Text) | 마이크 음성 (말하기 미션 사용 시) | 음성 → 텍스트 변환 |

## 5. 개인정보의 위탁

해당 사항 없음. (현재 별도의 자체 서버 인프라를 운영하지 않습니다.)

## 6. 정보주체의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

- 단말 내부 데이터는 앱 삭제로 즉시 파기됩니다.
- 구독 데이터 삭제는 회사 이메일로 요청하시거나 RevenueCat의 사용자 삭제 API를
  통해 처리됩니다.
- 구독 해지는 Google Play의 구독 관리 메뉴에서 직접 진행할 수 있습니다.

## 7. 개인정보의 안전성 확보 조치

- 회사는 자체 서버를 운영하지 않으므로, 외부 통신 구간은 모두 HTTPS로 암호화되어
  Google Play / RevenueCat에 전달됩니다.
- 구독 관리에 사용되는 API 키는 빌드 시점에 주입되며, 클라이언트 코드에 평문으로
  포함된 키는 RevenueCat의 공개 가능한 API 키(`appl_*`, `goog_*`)에 한정됩니다.
